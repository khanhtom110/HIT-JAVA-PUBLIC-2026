1. OOP trong Java

OOP (Object‑Oriented Programming) tổ chức chương trình quanh các đối tượng. Mỗi đối tượng có trạng thái (thuộc tính) và hành vi (phương thức). Ví dụ lớp Student có các thuộc tính id, name, score và phương thức getRank() để xác định xếp loại.

Bốn tính chất chính:

  • Đóng gói (Encapsulation)
  • Kế thừa (Inheritance)
  • Đa hình (Polymorphism)
  • Trừu tượng (Abstraction)

        Student s = new Student("SV01", "An", 8.5);
        System.out.println(s.getRank()); // Giỏi

2. Package, Class và Object

Package giúp tổ chức các class vào không gian tên logical.

package com.example.school;

public class Student {
    private String id;
    private String name;
    private double score;

    public Student(String id, String name, double score) {
        this.id = id;
        this.name = name;
        this.score = score;
    }

    public String getRank() {
        return score >= 8 ? "Giỏi" : score >= 5 ? "Đạt" : "Chưa đạt";
    }
}

Class là bản thiết kế; object là thực thể được tạo từ class.

5. Constructor, Getter, Setter

public Student(String id, String name, double score) {
    this.id = id;
    this.name = name;
    setScore(score);
}

public String getName() { return name; }

public void setName(String name) {
    if (name == null || name.isBlank())
        throw new IllegalArgumentException("Tên không được trống");
    this.name = name.trim();
}

public void setScore(double score) {
    if (score < 0 || score > 10)
        throw new IllegalArgumentException("Điểm phải từ 0 đến 10");
    this.score = score;
}

Constructor khởi tạo giá trị ban đầu; getter trả về giá trị; setter kiểm tra và gán giá trị.

Student s = new Student("SV03", "Cường", 9.0);
System.out.println(s.getRank()); // Giỏi
s.setScore(6.5);
System.out.println(s.getRank()); // Đạt
